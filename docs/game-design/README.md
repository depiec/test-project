# Game Design Document - Overlord-Inspired 2D Game

This directory contains the comprehensive game design document for our Overlord-inspired 2D game, which incorporates mechanics from Diablo, Dungeon Keeper, and Total War.

## Document Structure

### World Building
Contains lore, setting, races, factions, magic system, and world geography.

### Game Mechanics
Contains core gameplay, progression systems, mechanics from inspirations, and UI/HUD design.

### Technical Details
Contains technical specifications, engine choice, architecture, save/load systems, and performance requirements.

### Art Direction
Contains visual style, character design, environment art, UI/UX design, and art production pipeline.
